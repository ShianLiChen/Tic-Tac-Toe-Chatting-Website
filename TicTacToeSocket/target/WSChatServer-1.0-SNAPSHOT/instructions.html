Instructions

How to Use This Website

  1. Accessing Games/Chat Room
    • To access the Game, press the 'Play' button on the Home page
    • To access the dedicated Chat Room, simply go the Chat page in the top navigation bar
    • For both Game and Chat Room, in order to enter a room, press the 'ENTER' button with an empty 'Enter room code' field, this will create a new chat room. Alternatively, you can enter one of the codes under the 'List of Chat Rooms' to enter an existing chat room.
    • Note: Refreshing the page will make you leave the Room.
  2. Using the Chat
    • Once you've entered the chat room, you will be prompted to enter a username. Once you've done so, you're ready to chat!
  3. Playing the Game
    • Once you and your opponent have both entered the room and have both entered a username, you can start playing. The game is standard tic-tac-toe, so one player clicks a square on the game grid which fills it with either an X or an O. Players alternate until one user has won or there is a draw.
      Note: There is nothing preventing players from making multiple moves in a row so make sure that you are keeping track of whose turn it is.
    • To start a new game, all players must press the 'New Game' button in order to clear the grid. Not doing so will cause issues with displaying moves as well as the game's ability to read moves.
    • There is also the opportunity to play a 4x4 grid Tic Tac Toe game, by pressing the button below the Tic Tac Toe grid. You can switch between game modes (3x3 vs 4x4)
